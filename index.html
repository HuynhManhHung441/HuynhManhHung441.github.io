Project

Detection and classification of polyps in colonoscopy images

09-12/2022
Thesis report
Developer
In that thesis topic, I trained neural network models artificial for the purpose of supporting the work of detecting and classifying polyps in colonoscopy images of patients. This makes a lot of sense important in detecting and preventing colorectal cancer It is considered as one of the most dangerous diseases today. The TransResUNet network model is trained to detect position of the polyps by segmenting the areas of pixels that contain polyp. Model YOLOv7 trained to perform both polyp location detection tasks by zoning the bounding box for each polyp and classifying for the polyps found belonged to one of two categories labeled "adenomatous" or "hyperplastic".Besides, a simple interface program was also developed so that people use approach to the program to detect and classify polyps by model YOLOv7 and TransResUNet is more convenient.
  • Collect datasets
  • Training model by YOLOv7
  • Write report and presentation
  • Build UI by Flask
